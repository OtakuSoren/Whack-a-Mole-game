# Whack-a-Mole Game

A simple browser-based whack-a-mole game built with vanilla HTML, CSS, and JavaScript.

## Play locally

```bash
npm install
npm start
```

Then open `http://localhost:3000` in your browser.

## Deploy on GitHub Pages

1. Go to your GitHub repo and open **Settings** → **Pages**.
2. Under **Build and deployment**, set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. Click **Save**.
4. Wait 1–2 minutes, then open the URL shown on the Pages screen.

If you update the code, just push to `main` again and GitHub Pages will refresh.
